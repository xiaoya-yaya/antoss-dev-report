# 蚂蚁开源开发者年度数字报告

[![Open Source Report 2025](https://img.shields.io/badge/open_source_report-2025-yellow)](https://antoss-dev-report.open-digger.cn/ "Open Source Report 2025")

深夜一笔提交，远方一次回应。

今年，在蚂蚁的 **2,700** 多个仓库中，来自全球的 **8,673** 位开发者，一起变更了 **15,170,646** 行代码。

感谢你们成为蚂蚁开源的开发者，在开源和技术的世界里，共同推动着一次次微小而美好的改变。

还记得你今年在 GitHub 上的那些提交吗？在深夜的 bug report、change request...一行行可复用的改进，一次次高质量的协作，汇成我们共同的记忆与荣光。

**所有蚂蚁开源的开发者们，是时候开启 2025 年你在开源世界的数字回忆了。**
